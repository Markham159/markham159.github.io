      FAQ

      Practical stuff

      Is this canon?

      No. It is a fan-made personality quiz built for fun.

      Can I host it free?

      Yes. It is just static files, so services like GitHub Pages or Neocities are enough.

      Can I put it on a wiki?

      Usually you link to it from the wiki. Most wikis do not let regular pages run custom site-wide JavaScript and CSS freely.

      Can I add more characters later?

      Yes. The character and ability pools live in the JavaScript file, so expanding it is straightforward.
